Detailed Description

Renders text as a textured quad using SDL_ttf and OpenGL.

This class handles loading a text string into an OpenGL texture, managing its color, font size, alpha transparency, and rendering it on screen using your engine's transformation system.

It supports dynamic updates to text, color, font size, and alpha.
