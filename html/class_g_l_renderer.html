Detailed Description

Handles OpenGL-based rendering, including mesh drawing, shaders, viewport control, and camera logic.

This class encapsulates OpenGL initialization, shader handling, orthographic projection setup, texture caching, and rendering of 2D drawable objects using VBOs.
