\doxysubsection{Detailed Description}
Represents an Open\+GL shader (vertex or fragment) and handles loading and compilation. 

This class encapsulates the creation and management of a GLSL shader. It supports loading source code from a file, compiling it, and retrieving the shader ID used in Open\+GL programs.
