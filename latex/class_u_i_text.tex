\doxysubsection{Detailed Description}
Renders text as a textured quad using SDL\+\_\+ttf and Open\+GL. 

This class handles loading a text string into an Open\+GL texture, managing its color, font size, alpha transparency, and rendering it on screen using your engine\textquotesingle{}s transformation system.

It supports dynamic updates to text, color, font size, and alpha.
