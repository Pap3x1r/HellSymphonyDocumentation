\doxysubsection{Detailed Description}
Handles Open\+GL-\/based rendering, including mesh drawing, shaders, viewport control, and camera logic. 

This class encapsulates Open\+GL initialization, shader handling, orthographic projection setup, texture caching, and rendering of 2D drawable objects using VBOs.
