\doxysubsection{Detailed Description}
Manages the player\textquotesingle{}s current state and transitions between different states. 

The \doxylink{class_state_machine}{State\+Machine} handles runtime switching of player states (such as movement, attacking, etc.) by holding a pointer to the current state and allowing controlled transitions.
