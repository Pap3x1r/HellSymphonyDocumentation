\doxysubsection{Detailed Description}
UI button class with click functionality, hover detection, and optional label and slider. 

Extends \doxylink{class_textured_object}{Textured\+Object} and supports\+:
\begin{DoxyItemize}
\item Activation and enable/disable states.
\item Mouse hover detection.
\item Function callback on click.
\item Associated \doxylink{class_u_i_text}{UIText} label and \doxylink{class_slider_object}{Slider\+Object}. 
\end{DoxyItemize}
